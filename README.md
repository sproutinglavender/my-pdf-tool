# PDF Tool

A simple tool to convert PDFs, extract text, and handle images.

### Features:
- **Convert JPG to PDF**  
- **Convert DOCX to PDF**
- **Extract text from PDFs**
- **Extract images from PDFs**

### Usage:
1. Run the program.
2. Click the button you need.
3. Done.
